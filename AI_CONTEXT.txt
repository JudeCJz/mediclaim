Current Architecture
- Frontend: React + Vite in `/src`
- API client: Axios in `/src/api.js`
- App state: `/src/context/AppContext.jsx`
- Backend: Express + Mongoose in `/server`
- Auth: JWT via `x-auth-token`
- Realtime: Socket.io events `CLAIM_UPDATED`, `USER_UPDATED`, `FY_UPDATED`
- Storage: Multer uploads via `/uploads`
- Database: MongoDB

Run / Verification Notes
- Frontend dev server: `npm run dev`
- Backend dev server: `npm run backend`
- Backend env expected in `/server/.env`:
  - `PORT=5000`
  - `MONGO_URI=...`
  - `JWT_SECRET=...`
  - `UPLOADS_DIR=...`
- Optional SMTP env for real email sending:
  - `SMTP_HOST`
  - `SMTP_PORT`
  - `SMTP_USER`
  - `SMTP_PASS`
  - `SMTP_FROM`
- Without SMTP env, `/api/mail` currently accepts requests but only simulates queueing.
- Without installing `nodemailer`, mail routes will fail when real SMTP sending is attempted.
